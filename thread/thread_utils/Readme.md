2、关于条件变量与线程
/**
 * 在测试的时候发现了一个问题，线程处于等待状态下，主线程调用signal试图唤醒线程
 * 但是主线程在调用signal之后就马上结束，会出现当前线程的wait无法被唤醒的情况
 * 所以我做了适当的延时，确保线程被唤醒，可以成功的join
 */

virtual ~CThread() {
    if (isRunning()) {
        usleep(1000 * 0.1);
        quit();
        join();
    }
    if (m_cond) delete m_cond;
}
